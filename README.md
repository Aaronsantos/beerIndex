# Beer.Index

A Beer list app with React/Redux.

## Installation

Clone the repository and run command below to install node dependencies
```
npm install or yarn install
```
## Start
```
npm start or yarn start
```
## Test
```
npm test or yarn test
```
## Technologies
- React
- React Router
- Redux
- Redux Thunk
- Stylled-Components
- Axios

## API
[PunkApi](https://punkapi.com/)
